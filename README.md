## 数据结构和算法是什么？
 - 数据结构： 计算机存储，组织数据的方式，就像锅碗飘盆。
 - 算法：一系列解决问题的清晰指令，就像食谱
 - 程序 = 数据结构 + 算法
 - 数据结构为算法提供服务，算法围绕数据结构操作。
